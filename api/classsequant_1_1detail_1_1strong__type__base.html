template<typename T, typename Tag>
class strong_type_base¶

Base class for strong types.

This class is modeled after the strong_type class of the excellent type_safe library, but with a stripped-down design covering our intended use cases. Namely, this class:

  • provides implicit cast to the underlying type to make many uses less verbose once the API surface has been breached

  • provides comparability and max/min functions if the underlying type supports them

  • provides range semantics if the underlying type supports it

Normally this is used as the base for the strong type, with the CRPT-like use of the strong type as Tag: 

struct StrongInt : strong_type_base<int, StrongInt> {
  using strong_type_base::strong_type_base;
};
